# CodeComfy

Generate images and videos with ComfyUI directly from VS Code — prompt, poll, download, assemble without leaving your editor.

## Key Features

- **Image generation** — single HQ images via ComfyUI presets
- **Video generation** — short clips with FFmpeg frame assembly
- **Real-time status bar** — queued, generating, done progress
- **Cancel support** — stop any generation mid-flight
- **Structured logging** — full run metadata in .codecomfy/runs/
- **Safety limits** — max 15s duration, 450 frames for video

## Install / Quick Start

Install from [Releases](https://github.com/mcp-tool-shop-org/codecomfy-vscode/releases) (`.vsix` file), then configure your ComfyUI URL in settings.
